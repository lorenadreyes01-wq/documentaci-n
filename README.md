# 1. Introducción

Visual Studio 2022 es un entorno de desarrollo integrado (IDE) creado por Microsoft. Permite desarrollar aplicaciones de escritorio, web, móviles y servicios utilizando diferentes lenguajes de programación, especialmente C#.

---

# 2. Requisitos del sistema

Antes de instalar Visual Studio, se recomienda contar con:

* Windows 10 o Windows 11.
* Procesador de 64 bits.
* 4 GB de RAM como mínimo.
* 20 GB de espacio disponible.
* Conexión a Internet.

---

# 3. Descarga de Visual Studio

Para descargar Visual Studio:

1. Accede al sitio oficial de Microsoft.
2. Selecciona la versión Community.
3. Descarga el instalador.
4. Ejecuta el archivo descargado.

---

# 4. Instalación y configuración

## Paso 1: Abrir el instalador

Una vez descargado el instalador, ejecútalo con permisos de administrador.

## Paso 2: Seleccionar las cargas de trabajo

Marca la opción:

* .NET Desktop Development

Esta opción permite crear aplicaciones Windows Forms y aplicaciones de consola utilizando C#.

## Paso 3: Instalar

Haz clic en **Install** y espera a que finalice el proceso.
